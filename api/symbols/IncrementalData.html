Interface IncrementalData

Hierarchy

  • IncrementalData

Model.toIncrementalData 的输出对象的接口

该对象的所有属性都是可选的，因为仅当发生更改时才包括它们。例如，如果事务不包含对modelData的任何更改，则此接口的modelData属性将不包含在输出对象中

模型数据中的任何共享或循环引用都将在输出对象中包含的数据上进行维护。
